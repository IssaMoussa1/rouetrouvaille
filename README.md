# La Rouetrouvaille 🩷

Projet web de reconquête — 4 pages HTML + musique.

## Structure
```
intro.html       ← page 0 : lettre d'intro
index.html       ← page 1 : question Oui / Non
yes.html         ← page 2 : célébration
wheel.html       ← page 3 : roue de la chance
_music.js        ← musique partagée (Glue Song)
glue-song.mp3    ← fichier audio
```

## Mise en ligne (Netlify Drop)
1. Va sur https://app.netlify.com/drop
2. Glisse ce dossier entier dans la zone
3. Copie le lien généré et envoie-le !
